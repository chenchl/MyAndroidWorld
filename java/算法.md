#### 1.在一个二维数组中（每个一维数组的长度相同），每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序。请完成一个函数，输入这样的一个二维数组和一个整数，判断数组中是否含有该整数。

```java
public class Solution {
    public boolean Find(int target, int [][] array) {
        if(array == null)
            return false;
        int row = array.length;
        int col = array[0].length;
        if(row == 0 || col == 0)
            return false;
        if(target < array[0][0] || target > array[row-1][col-1])
            return false;
        int temp = row -1;
        int i = 0;
       	//矩阵法
        while((temp >= 0) && (i < col)){
            if(array[temp][i] > target) {
                temp--;
            } else if(array[temp][i] < target) {
                i++;
            } else {
                return true;
            }
        }
        return false;
    }
}
```

#### 2.请实现一个函数，将一个字符串中的每个空格替换成“%20”。例如，当字符串为We Are Happy.则经过替换之后的字符串为We%20Are%20Happy。

```java
public class Solution {
    public String replaceSpace(StringBuffer str) {
        int oldsum = str.length();
        int newsum = 0;
        int blockNum = 0;
        int i = 0;
        while(i<str.length()){
            if(str.charAt(i) == ' ') {
                blockNum++;
            }
            i++;
        }
        i = 0;
        int j = 0;
        newsum = oldsum+2*blockNum;
        char[] chars = new char[newsum];
        while(i< oldsum) {
            if(str.charAt(i) == ' ') {
                chars[j++] = '%';
                chars[j++] = '2';
                chars[j++] = '0';
            } else{
                chars[j++] = str.charAt(i);
            }
            i++;
        }
        return String.valueOf(chars);
       
    }
}
```

#### 4.输入某二叉树的前序遍历和中序遍历的结果，请重建出该二叉树。假设输入的前序遍历和中序遍历的结果中都不含重复的数字。例如输入前序遍历序列{1,2,4,7,3,5,6,8}和中序遍历序列{4,7,2,1,5,3,8,6}，则重建二叉树并返回。

```java
public class Solution {
    public TreeNode reConstructBinaryTree(int [] pre,int [] in) {
        if (pre == null || in == null) {
            return null;
        }
        if(pre.length == 0)
            return null;
        if (pre.length != in.length) {//长度必然是一致的这是前提
            throw new IllegalArgumentException("长度不一样，非法的输入");
        }
        TreeNode node = new TreeNode(0);
        for(int i =0;i<in.length;i++) {
            if(in[i] == pre[0]) {
                node.val = in[i];
                node.left = reConstructBinaryTree(Arrays.copyOfRange(pre,1,i+1)
                                                  ,Arrays.copyOfRange(in,0,i));
                node.right = reConstructBinaryTree(Arrays.copyOfRange(pre,i+1,pre.length)
                                                  ,Arrays.copyOfRange(in,i+1,in.length));
            }
        }
        return node;
    }
}
```

#### 5.用两个栈来实现一个队列，完成队列的Push和Pop操作。 队列中的元素为int类型。

```java
public class Solution {

    Stack<Integer> stack2 = new Stack<Integer>();
    
    public void push(int node) {
        stack1.push(node);
    }
    
    public int pop() {
        if(!stack2.isEmpty()) {
            return stack2.pop();

        while(!stack1.isEmpty()) {
```

#### 12.输入一个链表，输出该链表中倒数第k个结点。

```java
public class Solution {
    public ListNode FindKthToTail(ListNode head,int k) {
        ListNode p1 = head;
        ListNode p2 = head;
        //P1先走到正数的K节点
        while( k>0&&p1 != null) {
            p1 = p1.next;
            k--;
        }
        //如果不足K个节点或者只有1个节点则返回
        if(p1 == head ||(p1 == null && k>0)) {
            return null;
        }
        //P1 P2一起走 直到P1走到最后一个节点时 P2刚好走到倒数第K个节点
        while(p1 != null) {
            p1 = p1.next;
            p2 = p2.next;
        }
        return p2;
    }
}
```

#### 13.判断单链表是否成环算法

```java
private static boolean isCircle(Node head) {
        if (head == null) {
            return false;
        }
        //定义两个指针为同一起点
        Node n1 = head;   //慢指针
        Node n2 = head;   //快指针
        //只要有环的话，这个循环条件就绝对会满足，如果没有环的话，到了最后总不满足
        while(n2.next != null && n2.next.next != null) {
            n1 = n1.next; //n1一次走一步
            n2 = n2.next.next; //n2一次走两步
            if (n1 == n2) {  //如果成环，总会有一点n1==n2
                return true;
            }
        }
        return false;

    }
```

#### 15.单链表翻转

```java
public class Solution {
    //非递归
    public ListNode ReverseList(ListNode head) {
        ListNode temp;
        ListNode cur = head;
        ListNode pre = null;
        while(cur!= null) {
        	temp = cur.next;
        	cur.next = pre;
        	pre =cur;
        	cur = temp;
        }
        return pre;
    }
    //递归
    public ListNode reverseR(ListNode head){
        //空链表和一个结点的链表无需反转
        if(head == null || head.next == null){
            return head;
        }
        ListNode res = reverseR(head.next);
        //第二个结点的next指向第一个结点
        head.next.next = head;  
        //第一个结点的next指向null
        head.next = null;
        return res;
    }
}
```

#### 16.输入两棵二叉树A，B，判断B是不是A的子结构。（ps：我们约定空树不是任意一个树的子结构）

```java
public class Solution {
    //先序遍历一次 放入string 判断是否contains
    public boolean HasSubtree(TreeNode root1,TreeNode root2) {
        String a = preOrder(root1);
        if(a == null) {
            return false;
        }
        String b = preOrder(root2);
        if(b == null) {
            return false;
        }
        return a.contains(b);
    }
    //先序遍历
    private String preOrder(TreeNode root) {
        if(root == null) 
            return null;
        String str = "";
        Stack<TreeNode> stack = new Stack<>();
        while(root != null || !stack.isEmpty()) {
            if(root != null) {
                str += root.val;
                stack.push(root);
                root = root.left;
            } else {
                root = stack.pop();
                root = root.right;
            }
        }
        return str;
    }
}
```

#### 18定义栈的数据结构，请在该类型中实现一个能够得到栈中所含最小元素的min函数（时间复杂度应为O（1））。

```java
public class Solution {
    private Stack<Integer> stack = new Stack<>();
    private Stack<Integer> stackMin = new Stack<>();
    
    public void push(int node) {
        stack.push(node);
        if(stackMin.isEmpty()) {
            stackMin.push(node);
        } else{
            if(node < min()) {
                stackMin.push(node);
            }
        }
    }
    
    public void pop() {
        int num = stack.peek();
        stack.pop();
        if(num == min()) {
            stackMin.pop();
        }
    }
    
    public int top() {
        return stack.peek();
    }
    
    public int min() {
        if(stackMin.isEmpty())
            return -1;
        return stackMin.peek();
    }
}
```

#### 19.输入两个整数序列，第一个序列表示栈的压入顺序，请判断第二个序列是否可能为该栈的弹出顺序。假设压入栈的所有数字均不相等。例如序列1,2,3,4,5是某栈的压入顺序，序列4,5,3,2,1是该压栈序列对应的一个弹出序列，但4,3,5,1,2就不可能是该压栈序列的弹出序列。（注意：这两个序列的长度是相等的）

```java
public class Solution {
    public boolean IsPopOrder(int [] pushA,int [] popA) {
      LinkedList<Integer> stack = new LinkedList<>();
      int index = 0;
      for(int i = 0; i< pushA.length;i++) {
          stack.push(pushA[i]);
          //判断第二个序列是否能弹出
          while(!stack.isEmpty() && stack.peek() == popA[index]) {
              stack.pop();
              index++;
          }
      }
      //判断是否弹栈完毕 如果有剩余说明第二个序列中存在不符合条件的顺序 则为false
      return stack.isEmpty();
    }
}
```

#### 21.数组中有一个数字出现的次数超过数组长度的一半，请找出这个数字。例如输入一个长度为9的数组{1,2,3,2,2,2,5,4,2}。由于数字2在数组中出现了5次，超过数组长度的一半，因此输出2。如果不存在则输出0。

```java
import java.util.HashMap;
public class Solution {
    public int MoreThanHalfNum_Solution(int [] array) {
        int result=0;
        if(array == null || array.length == 0)
            return result;
        if(array.length == 1)
            return array[0];
        int size = array.length /2+1;
        //利用hashmap来记录每个数字出现的频率
        HashMap<Integer,Integer> map = new HashMap<>();
        for(int i = 0;i <array.length;i++) {
            if(map.containsKey(array[i])) {
                map.put(array[i],map.get(array[i])+1);
            } else{
                map.put(array[i],1);
            }
            if(map.get(array[i])>=size)
                return array[i];
        }
        return 0;
        
    }
}
```
